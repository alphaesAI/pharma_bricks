================================================================================
WORKFLOW (How JSON configs flow through the pipeline)
================================================================================

1. CONVERSION (Fixed-width → CSV)
   Input: submitted837_*.txt (from BizTalk)
   Config: JSON/Conversion/*.json
   Output: CSV files
   Used by: Conversion notebooks

2. PROCESSING (CSV → Parquet)
   Input: CSV files
   Config: JSON/Schema/*.json
   Output: Parquet files (Processed zone)
   Used by: MoveFileToProcess.scala

3. CONSOLIDATION (Parquet → Delta)
   Input: Parquet files
   Config: JSON/Consolidation/*.json + JSON/DataModels/*.json
   Output: Delta tables (Consolidated zone)
   Used by: MoveFileToConsolidation.scala

4. GOLD/PLATINUM (Delta → Enriched)
   Input: Consolidated Delta tables
   Config: JSON/Gold/*.json + JSON/Platinum/*.json
   Output: Gold/Platinum Delta tables
   Used by: GoldLayerProcessing notebooks
