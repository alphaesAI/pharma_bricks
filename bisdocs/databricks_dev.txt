SUBFOLDER: Notebooks/CommonMethods/ABC/
Purpose: Reusable code libraries and helper functions (imported by other notebooks)
------------------------------------------------------------------------------

ConversionMain.scala - Core conversion functions:
  * hasColumn - Checks if dataframe contains a column
  * lsplit - Creates a row of data from a List (fixed-width parsing)
  * getSortedSelect - Retrieves dynamic select list in specific order
  * createFixedWidthDF - Creates fixed width dataframe from file
  * createDelimitedDF - Creates dataframe using record type
  * removeLogFiles - Removes log files when writing to blob storage
  * removeFile - Removes single file from path
  * renameDF - Renames datafile
  * writeDfCSV - Creates CSV from dataframe
  * moveFile - Moves file from source to destination

CreateUserDefinedFunctions.py - Custom Spark UDFs:
  * overpunchUDF - Decodes overpunch financial notation (converts { to 0, } to -0, A to 1, etc.)

ErrorHandling.scala - Standard error response formatting for validation failures

FileHandling.scala - File operations:
  * isIgnoreHeader - Creates dataframe ignoring header
  * withoutHeader - Creates dataframe without header
  * delimitedFile - Creates dataframe from delimited file
  * path_exists - Checks if path exists (returns boolean)

GetTypes.scala - Data type conversion utilities:
  * getSQLType - Converts string scala datatype to SQL datatype
  * getDataType - Converts string scala datatype to true scala datatype
  * getStruct - Returns struct type from dataframe for building other dataframes

SynJSONCreatorClass.scala - Helper class to build JSON strings programmatically:
  * addBracketStart/End - Adds JSON array brackets
  * addBraceStart/End - Adds JSON object braces
  * addComma - Adds comma separator
  * addNewEntry - Adds key-value pair to JSON
  * getJSON - Returns final JSON string

ValidationsL3.py - Data validation rules (Level 3 validations):
  * rowEquals - Check if row equals expected value
  * rowGreaterEqual - Check if row >= expected value
  * rowCheckValidator - Main row check validator
  * fileRulesValidator - Main file check validator
